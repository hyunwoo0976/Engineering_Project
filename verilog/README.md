# My Verilog Design Project

## 1. Introduction

This repository is a structured project space focused on Verilog HDL, the foundational language for semiconductor design and verification. 

Rather than merely studying language syntax, this project aims to develop hands-on, industry-ready RTL design and verification skills. The ultimate objective is to reach a professional level of expertise capable of architecting, designing, and thoroughly verifying complex digital systems.

## 2. Learning Objectives

* **Mastery of Digital Logic Circuits**: Achieving a comprehensive understanding of digital logic design principles.
* **Advanced Testbench Development**: Enhancing verification skills by creating test scenarios that account for edge cases, and validating design integrity through rigorous waveform analysis.
* **Practical Development Environment Setup**: Building a simulation environment utilizing Icarus Verilog and GTKWave, and experiencing an automated compilation and verification pipeline via batch scripts (`.bat`).

## 3. Development & Verification Environment

* **Language**: Verilog 2001
* **Simulation & Waveform**: Icarus Verilog, GTKWave
* **Editor**: Visual Studio Code
